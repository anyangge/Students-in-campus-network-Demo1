# PC端学生校园网
>这是自己大二第二学期末做的一个web项目，使用到的技术有：div+css、jsp、Struts2、Hibernate、MySql

### 主要模块：
#### 论坛社区：提供看帖、发帖、回帖功能
#### 二手市场：提供产品发布、展示、搜索功能
#### 登录注册：实现用户帐号校验
